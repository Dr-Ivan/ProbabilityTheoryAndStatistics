## Тема 6. Создание собственных распределений в Python
- Задача: Рассмотреть функцию, которая является плотностью вероятности некоторой непрерывной случайной величины. Для этой случайной величины:
    + проверьте условие нормировки плотности;
    + постройте графики плотности и функции распределения;
    + рассчитайте вероятность попадания случайной величины в некоторый интервал;
    + вычислите математическое ожидание, дисперсию и среднее квадратическое отклонение; 
    + вычислите квантиль уровня q и p%-ную точку случайной величины;
    + определите коэффициент асимметрии и эксцесс.
- [Результат][topic6]
- [Описание][readme6]

## Тема 7. Исследование зависимости плотности от параметров НСВ в Python
- Задача: Повторить примеры с исследованием поведения графика плотности от значений параметров НСВ для равномерного, экспоненциального, нормального законов распределения и проверку формул расчета математического ожидания и дисперсии для сгенерированных данных. 
Исследовать поведение графика и расчет характеристик для другого вида непрерывного распределения. Для рассматриваемого распределения оформить теоретическую часть (пояснения и расчетные формулы).
