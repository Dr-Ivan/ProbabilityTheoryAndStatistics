# Моделирование связи частотного и классического определения вероятности

## Задача
В коробке находится 6 красных и 6 синих шаров. Случайным образом выбираем 6 шаров. Какова вероятность того, что среди них будет ровно 3 красных шара?

## Математическая модель

### Классическое определение
Вероятность вычисляется комбинаторно:

`P (3 красных шара из 6) = C(6,3) × C(6,3) / C(12,6) ≈ 0.4329`

где:
- C(6,3) = 20 - число способов выбрать 3 красных из 6
- C(6,3) = 20 - число способов выбрать 3 синих из 6  
- C(12,6) = 924 - общее число способов выбрать 6 шаров из 12

### Частотное определение
Вероятность оценивается через относительную частоту события при большом числе испытаний:

`P (3 красных шара из 6) ≈ (число успешных экспериментов) / (общее число экспериментов)`


## Как работает программа

1. **Вычисляет классическую вероятность** по комбинаторной формуле
2. **Проводит серию экспериментов** с увеличивающимся количеством испытаний (10, 50, 100, ..., 100_000)
3. **Строит график**, показывающий как частотная вероятность стремится к классической
4. **Выводит таблицу** с результатами всех экспериментов
