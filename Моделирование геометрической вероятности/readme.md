# Моделирование геометрической вероятности

## Задача
Метание дротиков в круг, площадь которого в два раза меньше площади квадратной стены.

## Математика 
_Расчет для фиксированных в коде параметров._ 
- Площадь квадрата со стороной 2: `S_квадрата = 2 * 2 = 4`
- Площадь круга радиусом 1: `S_круга = π * r^2 = π * 1 * 1 = π`
- Теоретическая вероятность: `P (попадания) = S_круга / S_квадрата = π/4 ≈ 0.7854`

## Как работает программа
1. Генерирует N случайных точек внутри квадрата
2. Считает, сколько точек попало в круг
3. Рисует красивую картинку с результатами
4. Выводит вероятности в консоль
