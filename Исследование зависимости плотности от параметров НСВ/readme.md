# Возможности программы

## 1. Исследование 4 типов распределений:
- **Равномерное распределение** - базовое распределение с постоянной плотностью
- **Экспоненциальное распределение** - модель времени между событиями
- **Нормальное распределение** - наиболее распространенное в природе
- **Бета-распределение** - гибкое распределение на интервале [0, 1]

## 2. Для каждого распределения отображается:
- График плотности вероятности с наложенной гистограммой выборки
- График функции распределения (теоретической и эмпирической)
- Автоматический расчет числовых характеристик
- Визуализация математического ожидания и стандартного отклонения

## 3. Интерактивные элементы:
- Выбор типа распределения из выпадающего списка
- Слайдеры для изменения параметров распределения
- Обновление всех графиков при изменении параметров и нажатии на кнопку


# Теория по распределениям

## 1. Равномерное распределение U(a, b)

**Область определения:** x ∈ [a, b]

**Плотность вероятности:**
```
f(x) = 1/(b-a),  при x ∈ [a, b]
f(x) = 0,        при x ∉ [a, b]
```

**Функция распределения:**
```
F(x) = 0,                    при x < a
F(x) = (x-a)/(b-a),          при a ≤ x ≤ b
F(x) = 1,                    при x > b
```

**Числовые характеристики:**
- Математическое ожидание: M(X) = (a + b)/2
- Дисперсия: D(X) = (b - a)²/12
- Стандартное отклонение: σ(X) = (b - a)/√12

**Параметры в программе:**
- `loc = a` - начало интервала
- `scale = b - a` - длина интервала


## 2. Экспоненциальное распределение Exp(λ)

**Область определения:** x ≥ 0

**Плотность вероятности:**
```
f(x) = λe^{-λx},  при x ≥ 0
f(x) = 0,         при x < 0
```

**Функция распределения:**
```
F(x) = 1 - e^{-λx},  при x ≥ 0
F(x) = 0,            при x < 0
```

**Числовые характеристики:**
- Математическое ожидание: M(X) = 1/λ
- Дисперсия: D(X) = 1/λ²
- Стандартное отклонение: σ(X) = 1/λ

**Свойство отсутствия памяти:**
```
P(X > s + t | X > s) = P(X > t)
```

**Параметры в программе:**
- `scale = 1/λ` - среднее время между событиями
- λ = 1/scale - интенсивность событий


## 3. Нормальное распределение N(μ, σ²)

**Область определения:** x ∈ (-∞, ∞)

**Плотность вероятности (функция Гаусса):**
```
f(x) = 1/(σ√(2π)) * e^{-(x-μ)²/(2σ²)}
```

**Функция распределения** не имеет простого аналитического выражения и вычисляется через интеграл:
```
F(x) = ∫_{-∞}^{x} f(t)dt
```

**Числовые характеристики:**
- Математическое ожидание: M(X) = μ
- Дисперсия: D(X) = σ²
- Стандартное отклонение: σ(X) = σ

**Свойства:**
- Симметричность относительно точки x = μ
- Точки перегиба: x = μ ± σ
- Правило 3-х сигм: P(|X-μ| < 3σ) ≈ 0.997

**Параметры в программе:**
- `loc = μ` - математическое ожидание (центр распределения)
- `scale = σ` - стандартное отклонение


## 4. Бета-распределение Beta(α, β)

**Область определения:** x ∈ [0, 1]

**Плотность вероятности:**
```
f(x) = x^{α-1}(1-x)^{β-1} / B(α, β),  при x ∈ [0, 1]
f(x) = 0,                             при x ∉ [0, 1]
```
где B(α, β) = Γ(α)Γ(β)/Γ(α+β) - бета-функция,
Γ(n) = (n-1)! - гамма-функция

**Функция распределения**:
```
F(x) = I_x(α, β) = ∫₀ˣ t^{α-1}(1-t)^{β-1}dt / B(α, β)
```

**Числовые характеристики:**
- Математическое ожидание: M(X) = α/(α + β)
- Дисперсия: D(X) = αβ/[(α+β)²(α+β+1)]
- Мода: Mo(X) = (α-1)/(α+β-2), при α, β > 1

**Специальные случаи:**
- При α=β=1: U(0, 1) - равномерное распределение
- При α=β: симметричное распределение
- При α=β=0.5: распределение арксинуса

**Формы распределения в зависимости от параметров:**
- α, β < 1: U-образное распределение
- α < 1 < β: J-образное (правостороннее)
- β < 1 < α: J-образное (левостороннее)
- α, β > 1: унимодальное (колоколообразное)
- α = β: симметричное относительно 0.5

**Практическое применение:**
- Моделирование вероятностей и пропорций
- Байесовская статистика (сопряженное априорное распределение для биномиального)
- Анализ процентных данных и долей
- Задачи оценки вероятности успеха

**Параметры в программе:**
- `a = α` - первый параметр формы
- `b = β` - второй параметр формы




# Структура интерфейса

## 1. Панель управления (верхняя часть):
- **Выбор распределения** - выпадающий список с 4 типами распределений
- **Параметры распределения** - слайдеры для изменения параметров выбранного распределения
- **Теоретическая информация** - текстовое поле с формулами и расчетами
- **Кнопка "Обновить графики"** - принудительное обновление визуализации

## 2. Область графиков (нижняя часть):
- **Левый график**: Плотность распределения и гистограмма выборки
  - Синяя гистограмма: эмпирическое распределение 
  - Красная линия: теоретическая плотность
  - Зеленая пунктирная линия: математическое ожидание M(X)
  - Желтая заливка: интервал ±σ (одно стандартное отклонение)
  
- **Правый график**: Функция распределения
  - Синяя линия: теоретическая функция распределения F(x)
  - Красная ступенчатая линия: эмпирическая функция распределения
  - Серые пунктирные линии: квантили уровней 0.25, 0.5, 0.75



# Методология исследования

## Генерация данных:
Для каждого распределения генерируется выборка из 1_000_000 случайных величин с текущими параметрами. Это позволяет:
1. Сравнить теоретическую плотность с эмпирической гистограммой
2. Проверить соответствие теоретических и выборочных характеристик
3. Визуализировать закон больших чисел

## Расчет характеристик:
- **Теоретические значения**: вычисляются по формулам на основе текущих параметров
- **Эмпирические значения**: вычисляются из сгенерированной выборки
- **Сравнение**: демонстрация сходимости эмпирических оценок к теоретическим

## Визуализация:
1. **Плотность распределения**: показывает форму распределения и влияние параметров
2. **Функция распределения**: демонстрирует вероятности и квантили
3. **Статистические метки**: 
    - M(X) - математическое ожидание
    - σ(X) - среднеквадратическое отклонение
